 # 🧮 Social Calculator

A sleek, responsive web-based calculator built with HTML5, CSS3, and Vanilla JavaScript. This was my very first project exploring DOM manipulation and event handling!

## 🚀 Live Demo
 https://kushal4175.github.io/Social-Calculator/

## ✨ Features
- **Clean UI:** Styled with a dark theme and "Coral" accents.
- **Keyboard Support:** Supports number keys, operators, Backspace, and Enter for calculations.
- **Advanced Constants:** Includes a dedicated button for **PI (π)**.
- **Input Validation:** Prevents common errors like leading zeros or consecutive operators (e.g., `++` or `*/`).
- **Responsive Layout:** Uses CSS Flexbox to ensure buttons align perfectly on different screens.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling using Flexbox and viewport units (`vw`/`vh`).
- **JavaScript (ES6):** 
  - `eval()` function for mathematical processing.
  - Event Listeners for keyboard interaction.
  - Logic to handle "Auto-focus" so you can type immediately.
